# On-Demand Subscription Demo

This repo hosts a demo application showing on-demand subscriptions powered by [Superfluid](https://superfluid.finance) money streams and [GoodDollar](https://www.gooddollar.org) G$ tokens. It is intentionally **vibe-coded**: most of the project was generated by AI agents using deep pre-research and near single-prompt continuous builds. Manual code contribution was kept under 5%.

## How it works

When a viewer presses play in the Video.js player, a G$ stream is started via Superfluid. Pausing, stopping or seeking beyond the preview adjusts or ends the stream.

## Repo layout

```
.
├─ apps/
│  └─ web/                 # Next.js app with Video.js wired
├─ packages/
│  ├─ streaming-sdk/       # Thin wrapper for starting/stopping streams
│  ├─ player-plugin/       # Video.js plugin gating playback
│  └─ ui/                  # Shared UI components
└─ docs/                   # Architecture notes and references
```

## Getting started

1. Install dependencies
   ```bash
   pnpm i
   ```
2. Build all packages
   ```bash
   pnpm -w turbo run build
   ```
3. Lint and type-check
   ```bash
   pnpm -w turbo run lint
   pnpm -w turbo run typecheck
   ```
4. Run the demo web app
   ```bash
   pnpm --filter @apps/web dev
   ```
